# Welcome to the ArcSpectra Wiki

**ArcSpectra** is a professional, high-performance tool designed for multispectral remote sensing in archaeology. This wiki serves as the complete documentation and user handbook.

### 📚 Documentation Sections

To get started, explore the detailed guides below:

1. **[🏛️ Archaeological Prospection Guide](Archaeological-Prospection)**
   Learn about the archaeological context of vegetation stress, how cropmarks and soilmarks occur, and how to select indices (NDVI, GNDVI, WDVI, etc.) to detect buried structures.
   
2. **[🎛️ User Interface Guide](User-Interface-Guide)**
   A complete reference directory of all input fields, dropdown menus, sliders, and tab views inside the ArcSpectra desktop dashboard.

3. **[🛠️ Sensor Configuration Guide](Sensor-Configuration)**
   Step-by-step mapping examples for common multispectral sensors like the **DJI Multispectral** and the **MicaSense Sequoia** using the Sensor Template Builder.
